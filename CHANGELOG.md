# Changelog

All notable changes to PokeRate will be documented in this file.

## [0.1.0.0] - 2026-04-14

### Added
- Generation filter: toggle Pokemon generations (I-IX) on/off to control which Pokemon appear in matchups
- Settings drawer accessible from navbar with Game Boy styled toggle buttons
- Filter indicator dot shows when not all generations are enabled
- Settings persist across sessions via localStorage
